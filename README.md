# 📊 Módulo de Reporte de Ventas Mensuales

## Descripción

El módulo **Reporte de Ventas Mensuales** permite al administrador consultar, analizar y visualizar las ventas realizadas en la plataforma EventPass mediante filtros de fecha.

El sistema permite seleccionar un **año** y un **mes específico** para generar un reporte consolidado de:

- Cantidad de entradas vendidas por evento.
- Ingresos totales generados.
- Recaudación mensual acumulada.
- Historial de ventas almacenado en el sistema.

Este módulo está integrado exclusivamente en el **Panel Administrativo**.

---

# 🚀 Características principales

## 🔎 Filtro por período

El administrador puede seleccionar:

- Año del reporte.
- Mes específico.
